mac下php redis 初体验

redis php-redis安装

redis安装

redis2.6 (已经不推荐使用了，不过我还是试了一下)

下载redis2.6
解压、安装

tar zxf redis-2.6.14.tar.gz
cd redis-2.6.14
make && make install

复制配置文件

sudo cp ~/redis-2.6.14/redis.conf /etc/

启动redis 服务

/usr/local/bin/redis-server /etc/redis.conf

连接redis

redis-cli -h 127.0.0.1 -p 6379
>就可以做set name 简单的操作了

php redis扩展安装

下载地址，使用说明 https://github.com/phpredis/phpredis

unzip phpredis-develop.zip 
cd phpredis-develop
/usr/local/php/bin/phpize
./configure --with-php-config=/usr/local/php/bin/php-config
make && make install

安装成功标志：
Installing shared extensions: /usr/local/php/lib/php/extensions/no-debug-zts-20131226/

在php.ini里面，引入扩展redis.so

extension = redis.so
[前提是指定了extension_dir]

成功标志: 运行 php -m | grep redis就能看见redis

redis.conf 配置文件

#是否允许后台运行，一般改为yes
daemonize: yes
#可用数据库数，默认值为16，默认数据库为0
databases 16
#保存数据到disk的策略
#当有一条Keys数据被改变是，900秒刷新到disk一次
save 900 1
#当有10条Keys数据被改变时，300秒刷新到disk一次
save 300 10
#当有1w条keys数据被改变时，60秒刷新到disk一次
save 60 10000
#当dump .rdb数据库的时候是否压缩数据对象
rdbcompression yes
#本地数据库文件名，默认值为dump.rdb
dbfilename dump.rdb
#本地数据库存放路径,默认值为 ./
dir ./

php使用redis

手册http://www.cnblogs.com/ikodota/archive/2012/03/05/php_redis_cn.html

例子：

$redis = new Redis();
$redis->connect('127.0.0.1', 6379);
$name = $redis->get('name');
var_dump($name);

redis升级

上官网下载redis-3.0.0

tar zxf redis-3.0.0.tar.gz
cd redis-3.0.0
make
//编译之后直接copy 5个程序到/usr/local/bin/ 覆盖以前的5个redis-* 程序
//注意不要吧.c .o 这个也复制过去
cp ./src/redis-* /usr/local/bin/
#php #redis
