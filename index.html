用hexo搭建的博客

首先这个博客已经筹备很久了，这几天才弄起来，实在是有些惭愧啊。以此纪念我的hexo搭建的静态博客。目前page托管在github上；

在搭建博客的过程中，首先感谢下面两个博主:

  • http://zipperary.com/categories/hexo/
  • http://blog.csdn.net/poem_of_sunshine/article/details/29369785/

1.hexo搭建

本地环境：mac、git、node

  • 安装hexo

    npm install -g hexo
hexo init
npm install

    安装完之后，本地启动服务

    hexo generate
hexo server

    这样在本地就可以通过http://0.0.0.0:4000查看了，hexo服务启动之后，不能关闭，否则本地无法参加查看

2.配置git托管page

  • 配置ssh 避免每次提交都输入git的账号密码
1. 查看是否有以前的key,有责备份

    `cd ~/.ssh`

    `mkdir key_backup`

    `mv id_rsa* key_backup`

2. 生成SSH key

    `ssh-keygen -t rsa -C "xxxx@xxxx.com”`

3. 登录到GitHub页面，Account Settings->SSH Public Keys->Add another key将生成的key(id_rsa.pub文件）内容copy到输入框中，save。

4. 测试链接

    `ssh git@github.com`

5. 设置git账号信息

    `git config --global user.name "treason258”`
    `git config --global user.email ma.tengfei2008@163.com`

6. 设置托管git的路径 编辑根目录_config.yml 【很重要以上5步都可以先不做】

        deploy:
              type: git
             repo: git@github.com:xxxx/xxxx.github.io.git
             branch: master
7. 提交至git

    `hexo generate`

    `hexo deploy`    

8. 这样就ok了，yidiandianlan.github.io可以查看

3.创建新的文档

hexo new "My New Post"

创建名为My-New-Post的项目
在source/_post下面生成My-New-Post.MD

4.配置自己的域名

在source目录下创建文件CNAME 写入 helonghua.com 即可

上传至git，再将helonghua.com 解析@.heonghua.com

  • 192.30.252.153

  • 192.30.252.154

这两个ip

5.常见问题

  1. 更换主题，下载之后放置theme下即可、修改_config.yml

    theme: yilia

  2. 快捷命令

    hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new

    快速提交上传git

    sudo hexo d -g
